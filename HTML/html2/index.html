A linguagem HTML

HTML (abreviação para expressão inglesa Hypertex Markup Language, que significa Linguagem de Marcção de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas Web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos Recursos da linguagem HTML, vistos até o momento:
Recursos Tags Associadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar lista de itens não ordenados
Imagens <img> Inserir imagens
